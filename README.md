## Lancer le projet

`npm install`  

`npm start`  

Aller sur : **localhost:8001**  

## Ajouter des slides

Ajouter une `<section>` dans **index.html**


## Code

Utilisation d'une grille simple (`.col2`) :  

```html
<section>
    <h2>Title</h2>
    <h3>Subtitle</h3>
    <div class="col2">
        <!-- first col -->
        <p>
            <img src="/images/some.gif" alt="">
            <i>Image description</i>
        </p>
        <!-- second col -->
        <ul>
            <li>some</li>
            <li>bullet</li>
            <li>points</li>
        </ul>
    </div>
</section>
```
