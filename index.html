Le concept de Flash

  • Macromedia lance Flash 1.0 en décembre 1996, Adobe rachètera la technologie en 2005

Les navigateurs web

Entre adoption d'une technologie et problèmes de support

  • Tous les grands acteurs du web (Google, Firefox, Microsoft...) font partie du W3C et font avancer le web
  • Ces groupes décident de l'évolution du HTML, de CSS, ou de Javascript
  • La communauté open-source joue également un rôle important
Le W3C (World Wide Web Consortium) décide en grande partie de l'évolution du web

Les navigateurs web

Un web fonctionnel pour tous

Chromium, la base de navigateur open-source créée par Google Can I Use informe les développeurs sur les incompatibilités entre navigateurs
  • Les navigateurs s'accordent pour permettre un web fonctionnel accessible à tous
  • Chromium sert de base à de nombreux navigateurs : Chrome, Opera, et maintenant Edge
  • Mais cette uniformisation pose aussi des soucis éthiques
